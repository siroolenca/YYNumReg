# YY接码注册

YY接码是一款方便快捷的验证码注册工具，用户可以通过YY接码快速获取手机验证码，用于注册各种社交平台、电商网站等需要手机验证的场景。

## 注册流程

1. 下载并安装YY接码APP
2. 注册并登录账号
3. 选择需要接收验证码的国家和手机号
4. 点击获取验证码，等待验证码到达并自动显示在YY接码APP上
5. 输入验证码完成注册
